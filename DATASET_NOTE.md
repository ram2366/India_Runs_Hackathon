# Dataset Note

The full `candidates.jsonl` dataset is not included in this GitHub-ready folder.

Reason:

- It is challenge-provided data.
- It is large and not suitable for a normal GitHub commit.
- The repository already contains the full code, labels, model artifacts, final CSV, audit files, and sample reference files needed to explain and reproduce the approach.

To rerun the project, place `candidates.jsonl` in the repository root and run the commands in `README.md`.
